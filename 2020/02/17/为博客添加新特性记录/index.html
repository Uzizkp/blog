添加FM页面增加音乐播放器

使用插件：hexo-tag-aplayer

这是一个Aplayer播放器的Hexo插件，引入 MetingJS 后，播放器将支持对于 QQ音乐、网易云音乐、虾米、酷狗、百度等平台的音乐播放，功能强大配置简单。

安装：

npm install --save hexo-tag-aplayer

在_cogig.yml配置：

aplayer:
    meting: true

在页面添加标签：配置详情

{% meting... %}
